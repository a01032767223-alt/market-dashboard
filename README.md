# 📊 48시간 시장 감시 대시보드

주식 시장의 자금이 어디로 흐르는지 실시간으로 추적하는 개인용 투자 대시보드입니다.

## 주요 기능

- **6개 지표 실시간 감시**: KOSPI/S&P500, BTC/ETH, 금 현물, BTC 도미넌스, VIX/공포탐욕, 환율/외국인수급
- **비트코인 4년 주기 타임라인**: 반감기 사이클에서 현재 위치를 자동 계산해 표시 (날짜 기반)
- **AI 투자 판단**: 지표 변화에 따라 매수/관망/경계 신호를 자동 재계산
- **5단계 하락장 매수 전략**: ATH 대비 -15%/-25%/-35%/-45% 트리거 자동 표시
- **실제 데이터 스파크라인**: 코인·금은 최근 7일 실제 가격 곡선 표시

## 색 규칙 (국내 관례)

- 🔴 **빨강** = 상승·과열
- 🔵 **파랑** = 하락·안정
- 🟡 **금색** = 안전자산(금) 전용

스파크라인 색은 7일 가격의 시작 대비 끝을 자동 판정하여 상승=빨강 / 하락=파랑으로 표시합니다.

## 데이터 출처

| 지표 | 출처 | 갱신 |
|------|------|------|
| BTC/ETH 가격 + 7일 곡선 | CoinGecko /coins/markets | 자동 |
| BTC 도미넌스 | CoinGecko /global | 자동 |
| 금 현물 (XAU) + 7일 곡선 | gold-api.com / freegoldapi | 자동 |
| 환율 (USD/KRW) | exchangerate-api | 자동 |
| KOSPI/S&P500, VIX, 공포탐욕, 외국인 | 수동 입력 | 하루 1회 |

1. 페이지를 엽니다 (자동 지표는 바로 갱신됩니다)

3. 상단 신호등과 AI 의견, 사이클 위치를 확인합니다
