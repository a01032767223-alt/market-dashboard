# 📊 48시간 시장 감시 대시보드

주식 시장의 자금이 어디로 흐르는지 실시간으로 추적하는 개인용 투자 대시보드입니다.

## 주요 기능

- **6개 지표 실시간 감시**: KOSPI/S&P500, BTC/ETH, 금 현물, BTC 도미넌스, VIX/공포탐욕, 환율/외국인수급
- **비트코인 4년 주기 타임라인**: 반감기 사이클에서 현재 위치를 자동 계산해 표시
- **AI 투자 판단**: 지표 변화에 따라 매수/관망/경계 신호를 자동 재계산
- **CORS 프록시 자동 갱신**: 직접 호출 → corsproxy.io → allorigins 순으로 시도
- **수동 보정**: 자동이 막히면 모든 자동 지표를 입력창에서 직접 보정 가능

## 자동 갱신 구조 (A+B 혼합)

GitHub Pages(https)에서 외부 API를 직접 부르면 CORS에 막히는 경우가 많습니다.
그래서 이 대시보드는 3단계로 시도합니다:

1. **직접 호출** — CORS를 허용하는 API는 그대로 성공
2. **corsproxy.io 경유** — 1이 막히면 프록시로 우회
3. **allorigins 경유** — 2도 막히면 다른 프록시로

세 방법이 모두 실패하면, 현재 실제 시세로 채워둔 **초기값**이 표시되고,
하단 입력창에서 직접 보정할 수 있습니다.

1. 페이지를 엽니다 (자동 갱신 시도)
2. 자동 지표가 이상하면 하단 입력창에서 보정 후 "저장하고 다시 판단"
3. 지수·VIX·공포탐욕·외국인은 개장 시 하루 1회 입력
4. 상단 신호등과 AI 의견, 사이클 위치를 확인
